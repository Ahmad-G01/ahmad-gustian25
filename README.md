## ahmad-gustian25 



## Clone this project

```bash
> git clone https://github.com/MhankBarBar/termux-wabot
```

## Install the dependencies:
Before running the below command, make sure you're in the project directory that
you've just cloned!!

```bash
> cd ahmad-gustian25
> bash install.sh
```

### Usage
```bash
> npm start
```
